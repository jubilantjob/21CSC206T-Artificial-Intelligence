Basics of AI

Artificial Intelligence (AI) focuses on designing intelligence in artificial devices, a term coined by John McCarthy in 1956. Intelligence is defined as the capability to acquire, understand, and apply knowledge to achieve worldly goals. AI studies mental faculties through computational models and treats intellectual processes as computational processes. An AI program demonstrates intelligence that equals or exceeds human intelligence in specific tasks. AI is a unique field that shares boundaries with Mathematics, Computer Science, Philosophy, Psychology, Biology, and Cognitive Science. It generally describes an attempt to build machines that can think, act, learn, and use knowledge to autonomously solve problems.

Definitions of AI

The definitions of AI are categorized based on measuring success against human performance versus rationality, and focusing on reasoning versus behavior.

  • Think Human-Like (Cognitive Science Approach): Focuses on making computers think like machines with minds, automating human thinking activities such as decision-making and learning. It requires a precise model for human cognition to produce a sequence of reasoning steps similar to a human.

  • Think Rationally (Laws of Thought Approach): The study of computational models that make it possible to perceive, reason, and act optimally. It formalizes the reasoning process using logical rules and inferences.

  • Act Human-Like (Turing Test Approach): Creating machines that perform functions requiring human intelligence, making computers do things that people currently do better. The Turing Test evaluates this by having a machine attempt to fool an interrogator into believing it is human.

  • Act Rationally (Rational Agent Approach): Emulating intelligent behavior via computational processes. It focuses on developing systems that act sufficiently, if not optimally, in all situations.

Strong AI claims computers can think on a level equal to humans, whereas Weak AI states that thinking-like features can be added to computers to make them useful tools.

History of AI

Important milestones that laid the groundwork for AI include:

  • 1920-30s (1931): Goedel laid the foundation of Theoretical Computer Science with the first universal formal language.
  • 1936: Turing reformulated Goedel’s result.
  • 1956: John McCarthy coined “Artificial Intelligence” at the Dartmouth Conference.
  • 1957: Newell, Shaw, and Simon demonstrated the General Problem Solver (GPS).
  • 1958: John McCarthy invented Lisp at MIT.
  • 1959: Arthur Samuel wrote the first checkers program capable of challenging a world champion.
  • 1963: Ivan Sutherland introduced interactive graphics with Sketchpad.
  • 1966: Ross Quillian demonstrated semantic nets.
  • 1967: The Dendral program interpreted mass spectra (the first successful knowledge-based program for scientific reasoning), and Doug Engelbart invented the mouse.
  • 1968: Minsky and Papert published Perceptrons.
  • 1972: Alain Colmerauer developed Prolog.
  • Mid-80s: Neural Networks gained popularity with the Backpropagation algorithm.
  • 1990: Major advances occurred across machine learning, data mining, NLP, and virtual reality.
  • 1997: Deep Blue defeated World Chess Champion Kasparov.
  • 2002: iRobot introduced the Roomba vacuum cleaner.

Foundations and Sub-Areas

AI foundations include Philosophy (issues of knowledge and belief), Psychology/Cognitive Science (problem-solving), Neuro-Science (brain architecture), Computer Science (algorithms, logic, inference), and Mathematics/Physics (statistical modeling).

Sub-areas and applications include: * Game Playing: Deep Blue beating Gary Kasparov. * Speech Recognition: PEGASUS interface for American Airlines. * Computer Vision: Face recognition, ALVINN autonomous driving system, and handwriting recognition. * Expert Systems: Diagnostic systems like MYCIN and Pathfinder, system configuration like DEC’s XCON, and financial decision-making for fraud detection. * Classification Systems: NASA’s systems for classifying stars and galaxies. * Natural Language Understanding: Translation of web pages and manuals. * Scheduling and Planning: DARPA’s DART system for logistics and manufacturing scheduling.

AI is applied across business, engineering, manufacturing, medicine, education, and space shuttle scheduling.

Building AI Systems and Intelligent Agents

Building intelligent systems involves three main components: 1. Perception: Experiencing the world through sensors like cameras or manual input of symptoms. 2. Reasoning: Inference, decision-making, and classification using an internal model (e.g., neural networks, Bayes networks). 3. Action: Interacting with the environment via actuators, such as steering a rover or synthesizing speech.

An Agent perceives its environment through sensors and acts upon it through actuators. A percept sequence is the complete history of an agent’s perceptual inputs. The agent’s behavior is defined by an agent function, which mathematically maps a percept sequence to an action, while the agent program is the concrete implementation running on the architecture. For example, in a vacuum-cleaner world, a reflex agent perceives its location and dirt status, choosing to move left, right, or suck up dirt.

Problem Solving and State Space Search

AI addresses both common-place mundane tasks (recognizing objects, navigating) and expert tasks (medical diagnosis, chess). Problem solving generates solutions from observed data, characterized by goals, objects, and operations.

A problem can be formulated as a state space search involving: * State: Specification of the values of all attributes of interest. * Initial State: Where the process starts. * Goal State: Acceptable solutions to the problem. * Operator/Actions: Rules that change one state into another, possessing preconditions and effects. * Path Cost: The step cost to execute actions.

Search Algorithms

Algorithm selection depends on completeness, optimality, time complexity, and space complexity.

Uninformed (Blind) Search

These algorithms use no problem-specific information to guide the search. * Breadth-First Search (BFS): Searches the goal along each breadth of the tree. It uses a NODE-LIST and adds new states to the end. BFS is complete and optimal (if all operators have the same cost), with time and space complexity of \(O(b^d)\). However, it requires exponential memory, making it ineffective for large search spaces. * Depth-First Search (DFS): Searches along the largest depth of the tree, adding new states to the front of the NODE-LIST. It takes exponential time \(O(b^N)\) and linear space \(O(bN)\). DFS is not complete as it may fail in infinite or cyclic spaces. * Iterative Deepening DFS (IDS): Combines BFS completeness and optimality with DFS memory efficiency. It searches iteratively with increasing depth limits. IDS is faster than BFS and DFS and is preferred for large search spaces with unknown solution depths.

Informed (Heuristic) Search

These algorithms use heuristic functions—feedback mapping states to numerical desirability weights—to guide the search efficiently. * Greedy Best First Search: Uses \(f(n) = h(n)\), expanding nodes based strictly on the heuristic estimate to the goal. For example, using straight-line distances to navigate Romania. * A* Search Algorithm: Uses \(f(n) = g(n) + h(n)\), where \(g(n)\) is the cost to reach the node and \(h(n)\) is the estimated cost to the goal. It maintains an OPEN priority queue and a CLOSED list of evaluated nodes. A* is complete, optimally efficient, and admissible if the heuristic never overestimates the actual cost. * **Iterative Deepening A* (IDA*):** Eliminates A*’s memory constraints by using depth-first iterations guided by an \(f(n)\) cost threshold. It is complete, optimal, and uses linear space relative to the solution depth.
